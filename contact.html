Gunakanlah selalu free open software.

Penulis menggunakan OS: linux Lubuntu, Browser: Mozilla Firefox dan Google Chrome, software editor html, css, js: Bluefish, software grafis dan image: Inkscape (untuk membuat logo, icon), GIMP (untuk mengedit foto/image digital).

Untuk latihan praktek HTML, CSS dan Javascript dapat dipraktekan langsung baik di OS Linux maupun Windows dengan browser bawaan masing-masing. Editor bisa menggunakan aplikasi Vim di OS Linux, aplikasi Notepad di OS Windows. Sedangkan untuk latihan praktek JQuery, harus mendownload filenya dulu.

Untuk latihan praktek PHP, Mysql penulis menggunakan LAMP (Linux Apache Mysql PHP) di OS Linux. Sedangkan untuk OS Windows bisa menggunakan WAMPSERVER.
