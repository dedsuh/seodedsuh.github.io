Belajar Membuat Website

Membuat website itu sebenarnya sangatlah mudah karena dasar website adalah HTML (sebagai struktur/bangunan website) dan CSS (untuk memperindah/mempercantik penyajian halaman/isi website), karena kedua hal tersebut bukan bahasa pemrograman, dengan hanya pengetahuan dasar tentang html dan css kita sudah bisa membuat website yang bagus.

Sedangkan Javascript (untuk tampilan website interaktif) yang merupakan bahasa pemrograman memang tidak semudah html maupun css dalam mempelajarinya, namun saat ini sudah banyak "Javascript Libraries" (pustaka javascript) seperti JQuery yang memudahkan kita dalam pemrograman javascript.

Untuk membuat website kita harus memahami dahulu tentang Internet dan Web, ISP, Domain, Hosting, serta software aplikasi perlengkapan kerja dalam pembuatan website seperti Browser (mozilla firefox, google chrome), Editor html, css, javascript (Bluefish).

Mulai membuat website dengan HTML5 sebagai struktur.

Mulai mempercantik penampilan halaman website dengan CSS3.

Mulai melengkapi halaman website secara interaktif dengan Javascript.

Gunakanlah selalu free open software.

Penulis menggunakan OS: linux Lubuntu, Browser: Mozilla Firefox dan Google Chrome, software editor html, css, js: Bluefish, software grafis dan image: Inkscape (untuk membuat logo, icon), GIMP (untuk mengedit foto/image digital).

Untuk latihan praktek HTML, CSS dan Javascript dapat dipraktekan langsung baik di OS Linux maupun Windows dengan browser bawaan masing-masing. Editor bisa menggunakan aplikasi Vim di OS Linux, aplikasi Notepad di OS Windows. Sedangkan untuk latihan praktek JQuery, harus mendownload filenya dulu.

Untuk latihan praktek PHP, Mysql penulis menggunakan LAMP (Linux Apache Mysql PHP) di OS Linux. Sedangkan untuk OS Windows bisa menggunakan WAMPSERVER.
